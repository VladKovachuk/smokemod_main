# Лог разработки (кратко, по коду)

Хронология **не из git**, а по реализованному функционалу в текущей кодовой базе.

## Реализовано

1. **Базовый мод** — `ExampleMod`, регистрация предмета и звуков, креатив.
2. **CigaretteItem** — предмет с прочностью, слот головы (`Equipment`), использование с анимацией лука.
3. **Звуки** — затяжка при начале использования, выдох в `finishUsing`; остановка затяжки при прерывании.
4. **Кулдаун** — только после полной затяжки (не при отмене).
5. **Модели** — переключение на `lit_cigarette` по NBT/предикату на клиенте.
6. **Частицы** — `ParticleTypes.CLOUD` при выдохе, спавн на сервере.
7. **NicotineManager** — уровень по UUID, убывание по тикам, урон и Wither при 0, S2C синхронизация.
8. **HUD** — строка `Lungs: X%` на клиенте.
9. **Расширяемость** — `getNicotineAmount()` и `addPuff(player, amount)` для будущих типов сигарет.

## Известные ограничения

- Уровень никотина **не сериализуется** в файл мира при перезапуске сервера (только in-memory `HashMap`).
- `fabric.mod.json` всё ещё содержит шаблонное описание и ссылку на `assets/modid/icon.png`.
