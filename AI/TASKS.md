# Задачи Smoke Mod

Состояние на момент анализа кода в репозитории.

## Сделано

- [x] Регистрация мода `smokemod`, предмет `cigarette`, звуков, вкладка креатива (Ingredients).
- [x] Сигарета: экипировка в слот головы, анимация использования как у лука, кулдаун после полной затяжки.
- [x] Звуки затяжки и выдоха; остановка звука затяжки при прерывании.
- [x] Прочность предмета (max 100, −20 за полное использование).
- [x] Модель «зажжённой» сигареты через NBT `lit` и предикат `smokemod:lit`.
- [x] Частицы дыма при выдохе (кастомные `CIGARETTE_CLOUD`, без гравитации).
- [x] Система продолжительного выдоха (ExhalationManager) на 3 секунды.
- [x] Реалистичная динамика дыма: рост размера и плавное исчезновение.
- [x] `NicotineManager`: уровень 0–100, убывание, урон при 0, синхронизация S2C.
- [x] Сохранение уровняа никотина на диске (`PersistentState`).
- [x] HUD: Графический индикатор лёгких вместо текста.
- [x] Расширяемость: `getNicotineAmount()` в `CigaretteItem`, `addPuff(player, amount)` в `NicotineManager`.

## В процессе

- Полировка баланса (скорость убывания, урон).
- Обновление `fabric.mod.json` (описание, иконка).

## Следующие задачи

- Дополнительные предметы (сигары и т.д.) через подклассы `CigaretteItem`.
- Убрать или использовать mixins по назначению.
- Тесты на сервере с несколькими игроками (синхронизация, баланс).

Подробные единичные задачи см. в папке `AI/TASKS/`.
