# Правила для ИИ при работе с Smoke Mod

## Общее

1. **Опираться на реальный код** — не выдумывать API Minecraft/Fabric, не описывать несуществующие файлы.
2. **Совместимость** — целевая версия **Minecraft 1.20.1** и **Fabric**; не подмешивать код под Forge/NeoForge без явного запроса.

## Код

- Писать **читаемый** Java: понятные имена, без лишних абстракций.


## Ресурсы

- Текстуры/модели: namespace **`smokemod`**, пути в `assets/smokemod/`.

## Документация

- Обновлять `AI/*.md` только если пользователь просит или задача существенно меняет поведение мода.
