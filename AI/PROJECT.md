# Smoke Mod — описание проекта

## Что это за проект

**Smoke Mod** — мод для **Minecraft 1.20.1** на **Fabric**. Он добавляет атмосферную механику курения: предмет «сигарета», звуки затяжки и выдоха, визуальные эффекты (дым), прочность предмета и систему «уровня в лёгких» (никотин), отображаемую на HUD.

Целевая аудитория — игроки, которым нужен **ролевой/реалистичный** контент, а не только декоративный предмет в руке.

## Цель проекта

- Дать игроку **глубокое взаимодействие** с курительными предметами: не только модель в руке, но и **звук, частицы, состояние персонажа**.
- В перспективе (по дорожной карте автора): шейдеры, больше предметов, выращивание табака и т.д. Текущий код — **этап основы + первые игровые системы**.

## Основные функции (по коду на сейчас)

| Область | Описание |
|--------|----------|
| **Предмет** | `smokemod:cigarette` — прочность 100, тратится по 20 за полную затяжку; можно надеть в слот головы (`Equipment` + `HEAD`). |
| **Использование** | ПКМ: анимация как у лука (`UseAction.BOW`), звук затяжки; полное использование — звук выдоха, частицы дыма, пополнение «лёгких», кулдаун 4 с. |
| **Модель** | NBT `lit`: после первого использования переключается модель на зажжённую (`lit_cigarette`), предикат `smokemod:lit` на клиенте. |
| **Звуки** | `sound_cigarette` (затяжка), `exhalation50` (выдох); при прерывании затяжки затяжка останавливается через `StopSoundS2CPacket`. |
| **Никотин / лёгкие** | Уровень 0–100 в `NicotineManager`: убывание со временем, пополнение при выдохе, урон при 0; синхронизация на клиент; HUD текстом `Lungs: X%`. |

## Технические ограничения

- Версия: **Minecraft 1.20.1**, **Java 17+**, **Fabric Loader + Fabric API**.
- Отдельной **базы данных** нет: состояние никотина хранится в памяти на сервере (`Map<UUID, Integer>`), без сохранения в мир при перезапуске (см. `ARCHITECTURE.md`).
