# 002 — HUD: полоска лёгких вместо текста

## Goal

Заменить текст `Lungs: X%` на наглядный индикатор (полоска или иконки), расположенный **над** полоской голода, с сохранением синхронизации с сервером.

## Requirements

- Использовать текущее значение `ExampleModClient.clientNicotineLevel` (не дублировать логику на клиенте).



## Files involved

- `src/client/java/com/example/ExampleModClient.java` — `HudRenderCallback`.
- Опционально: новые текстуры в `assets/smokemod/textures/gui/` (если решишь делать не только цветную полоску кодом).

## Definition of Done

- Визуально понятно заполнение 0–100%.
- Значение обновляется при курении и при убывании (как сейчас через пакет).
- Нет регрессий: креатив/спектатор ведут себя как задумано в `NicotineManager`.
